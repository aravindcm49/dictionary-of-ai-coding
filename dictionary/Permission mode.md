The permission-gating slice of an **agent mode** (see [[Agent mode]]) — which tool calls (see [[Tool call]]) trigger a permission request (see [[Permission request]]) and which run automatically. The original purpose of mode systems before harnesses (see [[Harness]]) started bundling behavioral instructions on top.

*Usage:* "It paused on every grep — totally killed the AFK run." "Loosen the permission mode for read-only tools, keep prompting on writes and shell. Most permission requests on a research session are noise."
