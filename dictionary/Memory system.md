A system that attempts to make an agent (see [[Agent]]) **stateful** (see [[Stateful]]) across **sessions** (see [[Session]]). Persists information into the **environment** (see [[Environment]]) during a session and reloads it into the **context window** (see [[Context window]]) at the start of future ones, so the agent carries continuity beyond the user clearing (see [[Clearing]]) the session.

*Usage:* "I keep having to re-tell it I'm on Postgres, not MySQL." "Wire up a memory system — write the stack to disk on the first turn, reload it at session start. The model itself is stateless; the memory layer fakes continuity."
