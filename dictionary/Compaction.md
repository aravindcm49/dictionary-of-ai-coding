A **handoff** (see [[Handoff]]) done in-memory: the previous **session** (see [[Session]])'s history is summarised and seeds a fresh session. Lossy — detail traded for headroom. Triggered manually by the user, or automatically (see [[Autocompact]]).

*Usage:* "Context's getting heavy and I still have the test pass to do." "Compact before you start — write what's load-bearing into the summary prompt so the new session keeps the schema decisions and drops the exploration."
